HH Weather API

The purpose of this webpage to showcase the visualization part of analyzing weather changes as you get closer to the equator.

To accomplish this analysis, we first pulled the data from OpenWeatherMap API to assemble a dataset on over 500 cities.

We used Matplotlib to Plot various aspects of the weather vs latitude, factoring temperature, cloudiness, wind speed, and humidity. This site provide the source data and visualizations created as part of the analysis.
